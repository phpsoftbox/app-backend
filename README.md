# App Backend

Минимальный backend‑скелет приложения на PhpSoftBox.

## Установка через create-project

```bash
composer create-project phpsoftbox/app-backend my-app
```

## Быстрый старт

1) Скопируй окружение:

```bash
cp .env.example .env
```

2) Установи зависимости:

```bash
composer install
yarn install
```

3) Запусти Vite:

```bash
yarn dev
```
